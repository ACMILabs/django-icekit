Welcome to GLAMkit!
===================

GLAMkit is a CMS designed especially for
Galleries, Libraries, Archives and Museums (the GLAM sector) and pretty much
any cultural institution. For a more extensive overview, see
http://glamkit.com.

This documentation covers the technical usage and API of GLAMkit.
