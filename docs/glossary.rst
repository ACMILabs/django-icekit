Glossary of terms
=================

.. glossary::
   :sorted:

   Rich content type
      A Django model that allows ```fluent-contents```-style rich content blocks
      to be used, for example "Page", "Press release", "Event".

   Content plugin
      A Django model that represents a 'block' of content that can be added to
      any rich content type, for example "Text", "Image", "Embedded Media".

      Content plugins are related to rich content types by a generic foreign
      key.
